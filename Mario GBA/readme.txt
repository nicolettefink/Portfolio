Mario : Nicolette Fink

This game is a single level of a mario game. Mario has 3 extra lives, so if
he dies a fourth time, the game is over and you will restart the level. If you
make it to the flag, you've completed the level and you will be able to replay it.

Controls:
Left/right on D-pad -> moves mario left and right

A -> jump

START -> will start the game from the title screen,
	 restarts the level at either the game over
	 or level complete screens

SELECT -> will go back to the title screen if in the
	  middle of the level, restarts the level
	  at either the game over or level complete
	  screens
